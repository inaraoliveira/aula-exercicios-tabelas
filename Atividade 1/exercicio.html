Tabela de Descrição
Tag Descrição
TABLE A tag TABLE representa uma tabela em HTML.
TH Cada cabeçalho é representado pela tag TH e estão sempre em negrito e centralizado.
TR A tag TR são as linhas da tabela.
TD Cada célula é representada pela tag TD e são sempre regulares e alinhadas à esquerda.
THEAD Podemos usar o THEAD para representar o cabeçalho.
TBODY Podemos usar o TBODY para representar o corpo da tabela.
TFOOT Podemos usar TFOOT para representar o rodapé da tabela.
ALIGN A tag ALIGN significa alinhar e precisa da informação do local onde você quer alinhar a tag.
BORDER O atributo BORDER adiciona uma borda na tabela.
CAPTION A tag CAPTION é usada para definir o título da tabela.
ROWSPAN Esse atributo é usado para mesclar determinada quantidade de linhas.
COLSPAN É usado para mesclar determinada quantidade de colunas.
WIDTH Determina a largura da área do conteúdo de um elemento
